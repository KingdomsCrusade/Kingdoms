# Kingdoms Plugin

Kingdoms Plugin is an indev plugin which is dedicated on **organizing kingdoms with its unique database table system**, providing server admins a **better visualization of each kingdom member counts and trends**, meanwhile **repaying kingdoms for their hard works**.

<!-- An image here -->

#### Unique database table system

This plugin **uses tables like spreadsheets to store kingdom properties and members data** thanks to the integration of MySQL database*.

**Note that this plugin will utilize a database of your server to store data.*

#### Better visualization

With the help of table-like database, you can **view every piece of the data in every aspect**!

**<u>Example :</u>**

- Every properties of every data in just a command
- Member counts of every kingdom
- Owner and mayor of every kingdom

#### Repaying kingdoms for hard works

To keep **engaging players to keep contributing** on their kingdoms, we **implemented features for kingdoms** which they can get benefits on. **The bigger the kingdom is, the larger the benefit is**.

- Kingdom chats, with integration to Discord
- Central system*
- More coming soon!

**Plannings for future release, subjected to change*

#### Buffs & Nerfs

Like any games, **to avoid any unfair and keeping the game environment stable**, we **connected every important variables** that are crucial to balancing the game **to a single configuration file**!
